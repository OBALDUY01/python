# Python · Конспект

Функция
map()

Применяет функцию к каждому элементу итерируемого объекта по одному и возвращает итератор с результатами.

# синтаксис
map(функция, итерируемый_объект)
функция
что применить к каждому элементу — int, str, lambda, def
итерируемый
список, строка, кортеж, генератор и т.д.
возвращает
объект-итератор — нужно обернуть в list()
Главный пример

Ввод чисел из строки

input() всегда возвращает строку. split() делит по пробелам — но элементы остаются строками, а не числами. map(int, ...) превращает каждую строку в int.

# классический паттерн
a = list(map(int, input().split()))
"12 123 4324 5235 121" ← input() возвращает строку
↓ .split() делит по пробелам
['12', '123', '4324', ...] ← список строк, не чисел!
↓ map(int, ...) int() к каждому элементу
↓ list(...) итератор → список
[12, 123, 4324, 5235, 121] ← настоящие int ✓
→ map() возвращает объект-итератор, а не список. Чтобы получить список — оборачиваем в list(). Можно обернуть в tuple() или set() — зависит от задачи.
Примеры

Разные случаи

map + int / str / float

Встроенные функции как оператор — самый частый случай.

map + lambda

Когда нужна простая операция без отдельной функции.

map + def

Когда логика сложнее одной строки.

map + два списка

Применяет функцию к парам элементов из двух итерируемых.

# преобразование типов
# строки → числа
nums = list(map(int, ['1', '2', '3', '4']))
print(nums)         # → [1, 2, 3, 4]

# числа → строки
strs = list(map(str, [10, 20, 30]))
print(strs)         # → ['10', '20', '30']

# строки → float
fl = list(map(float, ['1.5', '2.7', '3.0']))
print(fl)           # → [1.5, 2.7, 3.0]
вывод 
[1, 2, 3, 4]
['10', '20', '30']
[1.5, 2.7, 3.0]
# map + lambda
nums = [1, 2, 3, 4, 5]

squares = list(map(lambda x: x ** 2, nums))
print(squares)      # → [1, 4, 9, 16, 25]

doubled = list(map(lambda x: x * 2, nums))
print(doubled)      # → [2, 4, 6, 8, 10]
вывод 
[1, 4, 9, 16, 25]
[2, 4, 6, 8, 10]
# map + def функция
def add_exclaim(word):
    return word + '!'

words = ['привет', 'пока', 'круто']
result = list(map(add_exclaim, words))
print(result)       # → ['привет!', 'пока!', 'круто!']
вывод 
['привет!', 'пока!', 'круто!']
# map с двумя списками
a = [1, 2, 3]
b = [10, 20, 30]

result = list(map(lambda x, y: x + y, a, b))
print(result)       # → [11, 22, 33]
вывод 
[11, 22, 33]
Паттерны

Типичное применение в задачах

★ На соревнованиях и задачах map(int, input().split()) — один из самых частых однострочников.
# ввод одной строки чисел
a = list(map(int, input().split()))
# ввод: "5 10 15 20"  →  [5, 10, 15, 20]
# ввод нескольких переменных сразу
x, y, z = map(int, input().split())
# ввод: "3 7 12"  →  x=3, y=7, z=12
# матрица — несколько строк
n = int(input())
matrix = [list(map(int, input().split())) for _ in range(n)]
# ввод 3 строки по 3 числа  →  [[...], [...], [...]]
Сравнение

map() vs list comprehension

map()

list(map(int, nums))

list(map(
  lambda x: x**2, nums
))

comprehension

[int(x) for x in nums]

[x**2 for x in nums]
→ Используй map() когда функция уже есть — int, str, float. Это короче и читабельнее. List comprehension лучше когда нужно писать выражение вручную.
