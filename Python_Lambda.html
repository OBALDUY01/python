# Python · Конспект

Анонимные функции
lambda

Функции без имени, записанные в одну строку. Идеальны для коротких операций внутри map, filter и сортировок.

# анатомия lambda

lambda x, y : x + y
keyword
ключевое слово, объявляет анонимную функцию
аргументы
один или несколько, через запятую
двоеточие
разделяет аргументы и тело
тело / выражение
одно выражение, результат возвращается автоматически
Основы

Базовый синтаксис

Анонимная функция — функция без имени, записанная в одну строку. Результат выражения возвращается автоматически — return писать не нужно.

# базовый пример
# обычная функция
def square(x):
    return x ** 2

# то же самое через lambda
a = lambda x: x ** 2

print(a(10))     # → 100
print(a(5))      # → 25
вывод 
100
25
λ Lambda-функция — это полноценный объект. Её можно присвоить переменной, передать как аргумент или вернуть из другой функции.
Аргументы

Несколько аргументов

Lambda принимает любое количество аргументов через запятую — так же как обычная функция. Можно задавать значения по умолчанию.

# два аргумента
add = lambda x, y: x + y
print(add(3, 7))         # → 10
вывод 
10
# аргумент по умолчанию
power = lambda x, n=2: x ** n

print(power(5))          # → 25  (n=2 по умолчанию)
print(power(5, 3))       # → 125
вывод 
25
125
# три аргумента
volume = lambda l, w, h: l * w * h
print(volume(2, 3, 4))   # → 24
вывод 
24
Применение

Внутри map, filter, sort

Главная сила lambda — передача как аргумент в функции высшего порядка. Не нужно определять отдельную функцию ради одной операции.

map()

Применяет функцию к каждому элементу последовательности.

filter()

Оставляет только элементы, для которых функция вернула True.

sorted() / sort()

Задаёт ключ сортировки через параметр key=.

reduce()

Сворачивает последовательность в одно значение (из functools).

# map — применить ко всем элементам
nums = [1, 2, 3, 4, 5]

squares = list(map(lambda x: x ** 2, nums))
print(squares)            # → [1, 4, 9, 16, 25]
вывод 
[1, 4, 9, 16, 25]
# filter — только чётные
nums = [1, 2, 3, 4, 5, 6]

evens = list(filter(lambda x: x % 2 == 0, nums))
print(evens)              # → [2, 4, 6]
вывод 
[2, 4, 6]
# sorted — сортировка по второму элементу
pairs = [('banana', 3), ('apple', 1), ('cherry', 2)]

result = sorted(pairs, key=lambda x: x[1])
print(result)
# → [('apple', 1), ('cherry', 2), ('banana', 3)]
вывод 
[('apple', 1), ('cherry', 2), ('banana', 3)]
# reduce — сумма всех элементов
from functools import reduce

nums = [1, 2, 3, 4, 5]
total = reduce(lambda acc, x: acc + x, nums)
print(total)              # → 15
вывод 
15
Ограничения

Что нельзя делать

Lambda содержит только одно выражение. Присваивание, операторы if/else как блок, циклы for, while — всё это недоступно.

✗ Присваивание внутри lambda и другие операции, которые меняют аргумент — вызовут ошибку. Lambda только вычисляет и возвращает выражение.
# ошибки — так нельзя
# ❌ присваивание — SyntaxError
bad = lambda x: x = x + 1

# ❌ несколько строк — SyntaxError
bad = lambda x:
    return x + 1

# ❌ цикл — SyntaxError
bad = lambda x: for i in x: print(i)
# ✅ можно — тернарный оператор
# условие через тернарный оператор — ок
clamp = lambda x: x if x > 0 else 0

print(clamp(5))    # → 5
print(clamp(-3))   # → 0
вывод 
5
0
Сравнение

lambda vs def

lambda

lambda x: x ** 2

def

def square(x):
    return x ** 2
→ Используй lambda для коротких одноразовых операций внутри других функций. Для сложной логики — всегда def, это читабельнее.
# итог — типичный паттерн использования
students = [
    {'name': 'Алия',   'grade': 88},
    {'name': 'Данияр', 'grade': 95},
    {'name': 'Мадина', 'grade': 76},
]

# сортируем по оценке (по убыванию)
top = sorted(students, key=lambda s: s['grade'], reverse=True)

for s in top:
    print(s['name'], s['grade'])
вывод 
Данияр 95
Алия 88
Мадина 76
